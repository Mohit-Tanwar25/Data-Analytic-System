# ✨ Features

## 📈 Analytics Dashboard

- Interactive sales trend visualization
- Category-wise sales analysis
- Regional profit analysis
- Real-time KPI metrics

## 📂 Dataset Management

- Upload unlimited CSV datasets
- Store datasets in MySQL database
- Switch between saved datasets
- Duplicate dataset cleanup system

## 🎨 Modern UI

- Beautiful dark & light mode
- Responsive modern dashboard
- Custom CSS styling
- Interactive cards and charts

## 📄 PDF Export

- Download complete dashboard reports
- Export charts and KPIs to PDF
- Auto-generated business insights

## 🤖 Smart Insights

- Automatic business insights generation
- Top category detection
- Profitability analysis
- Negative profit transaction detection
